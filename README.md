# Project Manager

A modern full-stack web application for managing projects with user authentication and role-based access.

## Tech Stack

- **Frontend:** Next.js 16 (App Router), React, Tailwind CSS, Radix UI
- **Backend:** Next.js API Routes, Prisma ORM
- **Database:** PostgreSQL
- **Authentication:** NextAuth.js
- **Language:** TypeScript
